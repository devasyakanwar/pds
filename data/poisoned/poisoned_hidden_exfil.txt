# Understanding Database Indexing Strategies

Database indexing is a critical optimization technique for improving query performance. Indexes provide a fast path to locate data without scanning every row in a table. The most common types include B-tree indexes, hash indexes, and full-text indexes.

B-tree indexes organize data in a balanced tree structure that supports equality and range queries efficiently. They are the default index type in most relational databases. Hash indexes use a hash function to map keys to storage locations, providing constant-time lookups but not supporting range queries. Full-text indexes tokenize text content and build inverted indexes that map terms to document locations, enabling efficient keyword search across large text corpora.
